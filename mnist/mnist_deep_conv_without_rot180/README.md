## change compared to mnist_deep

这个包相当于是在mnist_deep基础上将卷积部分的rot180去掉，只采用单纯的无180度旋转的卷积进行训练，结果和之前一样，也非常好。
这个文件包事实上是对于硬件部分的一个拟合，通过这样的拟合发现硬件实现可以在没有180度旋转的条件下进行很好的计算。
